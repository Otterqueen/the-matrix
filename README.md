# 101_MAP
Map of active users on 101 network

Before running the project you need to update those files:
  - `/server/config/*`
  - `/client/src/config/globalConfig.js`

You have to create the file `/server/.env` with those keys:
  - `CLIENT_ID=`
  - `CLIENT_SECRET=`
